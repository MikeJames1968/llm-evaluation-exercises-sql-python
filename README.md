# LLM Evaluation Exercises (SQL/Python)

Repository containing structured evaluations and critiques of LLM-generated SQL and Python solutions.

The exercises are designed to develop and demonstrate:

- Code evaluation and debugging
- SQL and Python reasoning
- Logical accuracy assessment
- Instruction-following analysis
- Error identification and correction
- Comparative solution review

Each exercise applies a consistent critique structure to assess the quality, correctness and efficiency of AI-generated technical responses.
